Intro¶

Consumers are fickle, and it can be frustrating for a business owner when a promotion they thought would succeed ends up falling flat. Customer analysis can help remedy this through the process of learning more about the behaviors of customers in order to direct their marketing campaigns towards specific demographics.

In this tutorial I will walk you through the process of gathering, cleaning, representing, and testing a database of customers from a service that sells a variety of products and wants to find out what makes a customer likely to purchase their promotions. Our goal is to determine what factors make a customer most likely to accept a promotional offer and then use those factors to predict if a customer will accept the offer.

We will start by importing all of the necessary libraries needed for this analysis. Notable tools are: Pandas - A library that we will be using for its built in data structures that are good for managing large datasets like ours. Numpy - A library that we will be using for its support for large, multidimensional arrays and functions to work on said arrays. Matplotlib - A powerful plotting library that we will be using almost any time a graph is required. Seaborn - A library that we will be using to create and display heatmaps Sklearn - A machine learning library that will be using for our logistical regression

We will also read the data from a csv file that we downloaded from "https://www.kaggle.com/datasets/whenamancodes/customer-personality-analysis" into a pandas dataframe

Now that the dataset is cleaned and prepped, we want to find correlations between variables. Since the original dataframe is somewhat lacking, I created a sister dataset with other useful metrics that we want to check.

An explanation of the new metrics: ID - The customer's ID carried over from the old dataset Age - The customer's age at sign up, calculated by subtracting Dt_customer year from birth year Income - The customer's income carried over from the old dataset Total_Spent - The total spent on purchases made in the last two years Pct_X - The percentage of customer's total purchases taken up by X in the last two years Pct_Total - The total percentage of customer's income(in the last two years) spent on all categories AcceptedAny - Whether or not the customer accepted a promotion carried over from old dataset

We will then use df.corr() to get a correlation matrix of all the variables and use seaborn to display that in a more visually palatable format. As one can see, wine seems to have a negative correlation with all other products while, ironically, gold has a negative correlation with income. One important thing is that the total amount spent on all products correlates with whether or not someone accepted an offer. This means total_spent will be a useful metric for later.

#Creating additional dataset that contains several useful metrics for spending habits
proto_dataframe = {"ID": [], "Age": [], "Income": [], "Total_Spent": [], "Pct_Wines": [], "Pct_Fruits": [],\
                   "Pct_Meat": [], "Pct_Fish": [], "Pct_Sweets": [], "Pct_Gold": [], "Pct_Total": [], "AcceptedAny": []}

for index, row in cust_data.iterrows():
    #Adding ID and Income to new dataset
    proto_dataframe["ID"].append(row["ID"])
    proto_dataframe["Income"].append(row["Income"])
    
    #Calculating age and appending it to new dataset
    dates = row["Dt_Customer"].split("-")
    date = dates[2]
    proto_dataframe["Age"].append(int(date) - row["Year_Birth"])
    
    #Calculating purchases
    num_wine = row["MntWines"]
    num_fruits = row["MntFruits"]
    num_meat = row["MntMeatProducts"]
    num_fish = row["MntFishProducts"]
    num_sweets = row["MntSweetProducts"]
    num_gold = row["MntGoldProds"]
    total = num_wine + num_fruits + num_meat + num_fish + num_sweets + num_gold
    
    proto_dataframe["Pct_Wines"].append((num_wine / total) * 100)
    proto_dataframe["Pct_Fruits"].append((num_fruits / total) * 100)
    proto_dataframe["Pct_Meat"].append((num_meat / total) * 100)
    proto_dataframe["Pct_Fish"].append((num_fish / total) * 100)
    proto_dataframe["Pct_Sweets"].append((num_sweets / total) * 100)
    proto_dataframe["Pct_Gold"].append((num_gold / total) * 100)
    proto_dataframe["Pct_Total"].append(total / (row["Income"] * 2))
    
    proto_dataframe["Total_Spent"].append(total)
    proto_dataframe["AcceptedAny"].append(row["AcceptedAny"])

Purchase_dataset = pd.DataFrame.from_dict(proto_dataframe)
sns.heatmap(Purchase_dataset.corr())
